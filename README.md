# NodeJS Bira Backend

Getting start code for **NodeJS Bira Backend**.

## Prerequisite

- NodeJS
- VSCode
  - Extensions - Prettier, ESlint, EditorConfig
- MongoDB - Atlas
- VSCode
- Postman

## Technology

- NodeJS
- ExpressJS
- MongoDB(Mongoose)

## Getting Started : Windows

- After cloning/forking

```sh
cd nodejs-bira-backend

npm install

npm start
```
